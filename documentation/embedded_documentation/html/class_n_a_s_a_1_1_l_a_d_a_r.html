DWORD NASA::LADAR::LADAR::DetermineBaudRate ( )

Tries to determine at what baud rate the LADAR is currently running by requesting scans at each possible rate. Returns the first rate at which a response is received.

DWORD NASA::LADAR::LADAR::TestSingleScan ( )

Tries to request a single scan from the LADAR, but ignores the data and instead returns 1 if it gets any response and 0 if it times out.
